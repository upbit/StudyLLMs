传统范式：Sense -> Plan -> Act

**SLAP范式**：
* S: Searching
* L: Learning
* A: Action
* P: Planning

慢思考：**Planning** -> Learning
快思考：**Searching** <--> **Action**, （Learning来完善Action）

### 机器狗训练数据收集

基础层训练（习惯，快思考）：
1. 动捕狗的动作
2. IK重定向到关节
3. 训练模型产生点击力矩参数
4. 通过强化学习，奖励动作最符合的情况（如走路，跑步等等）
5. 去掉Encoder，随机选取离散 latent code（探索效率高）
6. 真机部署测试（训练阶段随机化：摩擦力，电机力矩极限、随机推力，感受到的姿态+noise等）

高层训练（慢思考，如规划、人类操控）：单独层来完成不同的任务
